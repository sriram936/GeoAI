# GeoAI
This repo implements RandLANet with pretrained weights, on point cloud aerial lidar data of villages to classify ground and non ground points and finally generate DTM of the village

⚠️ The project is still under development

[Link to Kaggle](https://www.kaggle.com/code/mahik9969/geoai)

[Link to GeoAI custom dataset](https://www.kaggle.com/datasets/sriramm932/tera-baap-tera13231)
